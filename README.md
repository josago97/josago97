## El evento histórico de hoy
El 20 de marzo de 1987, Microsoft lanzó Excel para Windows, su poderosa hoja de cálculo que ayudó a consolidar el dominio de Windows en el mercado empresarial y de oficina.

## El temazo de hoy
#### [Depeche Mode - Enjoy the Silence](https://www.youtube.com/watch?v=aGSKrC7dGcY)
Publicado como single el 20 de marzo de 1990, este icónico tema de Depeche Mode se convirtió en uno de los mayores éxitos de la banda. La canción combina melodías electrónicas cautivadoras con letras profundas que reflexionan sobre la comunicación y la soledad, representando la cumbre del sonido del synth-pop de finales de los 80s.
