## El evento histórico de hoy
El 4 de julio de 1996, nació el proyecto open source Apache HTTP Server, uno de los servidores web más importantes de la historia, a partir de una bifurcación del código de NCSA HTTPd.

## El temazo de hoy
#### [Madonna - Borderline](https://www.youtube.com/watch?v=rSaC-YbSDpo)
Este éxito de 1984 de Madonna explora las complejidades de una relación amorosa inestable. La canción, con su característico sonido new wave, se convirtió en uno de los primeros grandes hits que ayudaron a lanzar su carrera al estrellato internacional.
