## El evento histórico de hoy
El 22 de abril de 1997, se lanzó la versión 4.0 del navegador Netscape Communicator, una suite de internet pionera cuyo código fuente luego se liberó, dando origen al proyecto Mozilla.

## El temazo de hoy
#### [Madonna - American Pie](https://www.youtube.com/watch?v=X4RDB-mMDJg)
Versión de Madonna del clásico de Don McLean, lanzada como single principal de la banda sonora de 'The Next Best Thing'. La canción mezcla el folk rock original con un estilo pop dance característico de la artista, generando tanto elogios como controversia por su reinterpretación del tema icónico.
