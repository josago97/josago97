## El evento histórico de hoy
El primer borrador de la especificación del lenguaje de programación **Rust** se publicó el 17 de noviembre de 2010, un hito clave para este lenguaje que ahora es fundamental para proyectos críticos como el núcleo de Linux y Firefox.

## El temazo de hoy
#### [The Rolling Stones - Rock and a Hard Place](https://www.youtube.com/watch?v=7pYBQg4qifU)
Lanzada como single el 17 de noviembre de 1989, esta canción del álbum 'Steel Wheels' combina el rock clásico de los Stones con un sonido más moderno de la época. La canción critica la situación política global de fines de los 80s, manteniendo el estilo rebelde característico de la banda.
