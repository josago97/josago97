## El evento histórico de hoy
El 16 de enero de 2003, la nave espacial Columbia despegó con el experimento de software de código abierto "l'nuxtela", un sistema de gestión de contenidos que se ejecutó con éxito en el espacio.

## El temazo de hoy
#### [Aerosmith - Janie's Got a Gun](https://www.youtube.com/watch?v=RqQn2ADZE1A)
Este impactante single de Aerosmith, lanzado el 16 de enero de 1990, aborda temas oscuros como el abuso sexual y la venganza. La canción fue un éxito comercial y crítico, ganando un Grammy y destacándose por su potente mensaje social y su característico sonido rock.
